## Principless

- **Customization**: Rising UI components are designed to be highly customizable, allowing you to modify styles, behaviors, and layouts easily.

- **Composition**: Components share a common interface, making them predictable and easy to integrate into your existing design system.

- **Accessibility**: All components are built with accessibility in mind, ensuring compliance with WCAG 2.0 standards.

- **Performance**: Components are optimized for high performance, ensuring a smooth user experience.

## Features

- **Pre-built Components**: Rising UI includes a variety of pre-designed components such as buttons, inputs, navigation menus, and more.

- **Theme Customization**: Easily customize themes using a theme editor or by overriding default styles.

- **Cross-Framework Support**: Compatible with popular frameworks like Next.js, Gatsby, and Vite.

## Contribute

It is an open-source project, and we welcome contributions from the community. If you have any ideas, suggestions, or bug reports, please feel free to open an issue or submit a pull request on [GitHub](https://github.com/sohomofficial/rising-ui).
